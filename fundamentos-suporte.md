# Fundamentos de Suporte de TI

## O que é suporte técnico?

Suporte técnico é a área responsável por ajudar usuários a resolver problemas relacionados a computadores, sistemas, redes, contas, impressoras, internet e ferramentas digitais.

## Exemplos de problemas comuns

- Computador lento
- Falha de conexão com a internet
- Esquecimento de senha
- Problemas com impressora
- E-mail não sincronizando
- Sistema não abrindo

## Processo básico de atendimento

1. Ouvir o usuário
2. Identificar o problema
3. Fazer perguntas objetivas
4. Testar possíveis causas
5. Aplicar solução
6. Registrar o atendimento
7. Confirmar se o problema foi resolvido
